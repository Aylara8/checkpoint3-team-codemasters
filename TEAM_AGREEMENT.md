# Git Workflow

1. Main branch must always stay stable
2. Each member creates their own branch

Branch naming format:

yourname-feature

Example:

aylar-add-form

Workflow:

1. Pull latest main
2. Create new branch
3. Make changes
4. Commit
5. Push branch
6. Create Pull Request
7. Teammate reviews
8. Merge into main
